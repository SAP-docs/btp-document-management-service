# Setting Up a Google Workspace Account

Create your Google Workspace Account to connect to Document Management Service, Integration Option.



<a name="loio9670f691dc974c5098cdeed2b73605ad__prereq_vkm_wws_h5b"/>

## Prerequisites

-   You've created a [Google Workspace](https://workspace.google.com/signup/businessstarter/welcome) business account by providing all the necessary details.

-   You've created a project in the [Google API Console](https://console.cloud.google.com/cloud-resource-manager?pli=1). For more information on the detailed steps, see [Creating a Project](https://cloud.google.com/resource-manager/docs/creating-managing-projects#creating_a_project).
