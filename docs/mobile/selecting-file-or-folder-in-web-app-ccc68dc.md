# Selecting File or Folder in Web App

You can use your Document Management Service desktop app to display any item from the *Document Management Service* folder in the Web app.



## Procedure

1.  Open the *Document Management Service* folder in your Explorer \(Windows\) or in Finder \(Mac\).

2.  Select the file or folder you want to display in the Web app.

3.  Choose *Select in Web App*.




## Results

The Web app opens and displays the selected file or folder.
